Comprender los conceptos básicos al desarrollo correspondiente al diseño de sitios web, implemetando las herramientas necesarias para la creación de contenidos en formato web.
OBJETIVOS: Diseñar un OVI acerca del curso de Diseños de sitios web, mostrando sus diferentes entornos y sus funcionalidades.
Objetivos especificos: - Investigar sobre el uso de GIT y GITHUB para la gestión de proyectos colaborativos de diseños web. - Conocer el uso de HTML5 y CSS3 para la creación de contenidos en formato web (páginas y sitios web). - Utilizar editores de código para HTML5 y CSS3.
